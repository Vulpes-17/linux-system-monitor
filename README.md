# Linux System Monitor

A lightweight Python command-line application that displays real-time Linux system health information.

## Features

- Displays hostname and operating system
- Reports CPU utilization
- Reports memory utilization
- Reports disk usage
- Displays system uptime
- Lists the top memory-consuming processes
- Calculates overall system health status

## Technologies

- Python
- psutil
- Linux

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
